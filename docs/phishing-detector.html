📮 Phishing Detector

Project Overview

"Phishing Detector" is a tool for analyzing and automatically responding to phishing emails. It extracts metadata, checks links in real-time, and applies categorization rules to support SOC teams.

Key Features

  • 📬 Email Parsing: Extracts headers, links, and attachments
  • 🧠 ML Classifier: Assesses phishing likelihood using machine learning
  • 🛡️ Threat Intelligence: Cross-checks domains and IPs against IOC databases
  • ⚙️ SOAR Ready: Supports automated responses through SOAR platforms

Stack

  • Frontend: HTML + CSS (neon theme)
  • Backend: Python (Flask)
  • Libraries: BeautifulSoup, Scikit-learn, VirusTotal API

Use Case

Ideal for SOC teams looking for a fast way to detect, triage, and respond to phishing attacks with automated evidence collection.
