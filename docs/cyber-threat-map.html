🔍 Cyber Threat Map

A real-time visualization tool that displays ongoing cyber attacks.

Features

  • 🌍 Geolocation of attack sources and targets
  • 🧠 TTP tagging based on MITRE ATT&CK
  • 📊 Visual data from threat intelligence feeds
  • ⚙️ Integration with SIEM/ELK

Tech Stack

HTML, CSS, JS, D3.js, APIs (AlienVault OTX, MISP)
